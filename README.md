# ⚙️ SmartFix Tanzania

Device repair booking platform for Tanzania.

## Quick Start
- See DEPLOYMENT_GUIDE.md for full step-by-step instructions
- Frontend: /frontend/App.jsx (React/Next.js)
- Backend: /backend/main.py (FastAPI/Python)

## Stack
- Frontend: React / Next.js → Vercel
- Backend: FastAPI (Python) → Railway
- Database: PostgreSQL → Supabase
- Payments: Flutterwave (M-Pesa) + Stripe (Cards)
- SMS: Africa's Talking
